# Java + GeminiAPI

Stack: Java, Springboot, JDBC, H2 Database, Env, Thymeleaf

## Configuring

Add your Gemini API Key to a .env file in the root of the project (Java-Gemini-QuestionMaker/poo/poo/).

```env
GEMINI_API_KEY=YOUR_API_KEY
```

## Dependecies

Inside the project file execute the following code in your terminal.
```
mvn clean install
```

## How to execute
Inside this diretory (Java-Gemini-QuestionMaker/poo/poo/src/main/java/com/univille/poo/)

Execute QuestionMakerApplication.java

## Usage

In the input field, write the topic you want to have the questions created for.
